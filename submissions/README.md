# Submissions a Kaggle

Este directorio contiene todos los archivos CSV enviados a Kaggle.

## Nomenclatura

Los archivos deben seguir el formato:
```
submission_XX_descripcion.csv
```

Donde:
- `XX`: Número de experimento (01, 02, 03...)
- `descripcion`: Breve descripción del modelo/técnica

## Ejemplo
- `submission_01_baseline_lr.csv` - Logistic Regression baseline
- `submission_02_rf_default.csv` - Random Forest con parámetros por defecto
- `submission_03_xgb_tuned.csv` - XGBoost con hiperparámetros optimizados
